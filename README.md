## Description

LabWork1 - Loading and processing a bitmap image

Image - a purely abstract class with an interface for image manipulation
BMPHeader - a class for storing and converting image titles in BMP format
BMPImg - bitmap image class and interaction

## Build

make
make tests

## Test command

./tests

## Run
./lab
<input\_file.bmp>

### Example:

sample.bmp

## Output

The program will generate:

`rotated_cw.bmp` (Rotated CW)

`rotated_ccw.bmp` (Rotated CCW)

`filtr.bmp` (Gauss Filtr)
